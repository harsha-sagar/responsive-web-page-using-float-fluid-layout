## Run Locally

Clone the project

```bash
  git clone git@github.com:harsha-sagar/responsive-web-page-using-float-fluid-layout.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
